🍽 지금 땡겨

"오늘 뭐 먹지?" 고민을 단숨에 해결합니다.

지금 땡겨는 사용자의 현재 시간, 날씨, 위치 등 맥락 정보를 바탕으로,
AI가 그 순간 가장 어울리는 메뉴를 추천해주는 음식 결정 도우미입니다.

✅ 핵심 기능

  • AI 맞춤 추천: GPT와 Claude 두 개의 AI를 활용하여 사용자 상황에 꼭 맞는 메뉴 추천
  • 원터치 추천: 앱 실행 후 단 한 번의 터치로 메뉴 추천 완료
  • 회원가입 없이 사용: 불필요한 절차 없이 누구나 즉시 사용 가능
  • 추천 이유 제공 (Claude 기반): 추천된 메뉴가 왜 지금 어울리는지 간단한 설명 제공

🧠 이렇게 작동합니다

  • 시간대 분석: 아침/점심/저녁/야식/간식 구분
  • 날씨 감지: 비, 더위, 추위 등 날씨별 맞춤 메뉴
  • 위치 기반: 사용자의 위치에 따른 메뉴 유형 변화
  • 기록 비저장 방식: 개인정보 수집 없이도 높은 정확도 구현

🎯 주요 대상

  • 매일 식사 결정에 고민하는 1인 가구
  • 시간에 쫓기는 직장인, 학생
  • 새로운 메뉴를 쉽게 발견하고 싶은 사람

📱 앱 정보

  • 출시일: 2025년 7월
  • 운영체제: iOS
  • 지원 언어: 한국어
  • 이용 방식: 무료 / 회원가입 없음
  • 다운로드: App Store에서 "지금 땡겨" 검색

🛠 기술 요약

  • AI: GPT (OpenAI), Claude (Anthropic)
  • 프론트엔드: Swift (iOS)
  • 백엔드: Firebase Functions, Node.js
  • 기타: WeatherKit, CoreLocation, Google Cloud Run

🔒 개인정보 및 보안

  • 회원가입/로그인 없음
  • 기기 정보 및 위치 외 별도 개인정보 수집 없음
  • 모든 요청은 익명으로 처리되며 저장되지 않음
